# 蓝航图书馆 React Native 前端重设计

## 目标

将 React Native 版前端全面改造为 Vue uni-app 版本的视觉风格，包括渐变导航栏、航空主题装饰、卡片式布局、统计行等设计元素，并补齐缺失的功能页面。

---

## 四、技术方案

### 4.1 装饰组件抽离

新建 `src/components/` 共享组件：

- `GradientHeader.tsx` — 渐变导航栏 + 弧线 + 可选装饰
- `SectionBar.tsx` — 蓝色竖条 + 标题 + 可选链接
- `StatCard.tsx` — 统计数字卡片（图标 + 数字 + 标签）
- `QuickEntry.tsx` — 快捷入口卡片
- `BookCoverBlock.tsx` — 渐变色块图书封面
- `MenuList.tsx` — 个人中心菜单项
- `ArcCurve.tsx` — 弧线装饰

### 4.2 渐变实现

由于 React Native 不支持 CSS `linear-gradient`，使用方式：

1. `expo-linear-gradient` 或 `react-native-linear-gradient`（已预装）
2. `<LinearGradient colors={['#1a3350', '#1e3a5f', '#2d5a87']} ...>` 实现导航栏

### 4.3 SVG 装饰

纸飞机、虚线圆环等使用 `react-native-svg`：

- `<Circle strokeDasharray="4,4" ...>` 虚线圆环
- `<Path>` 纸飞机图标

### 4.4 数据策略

- 有后端 API 的优先调用 API（现有 client.ts + auth.ts + books.ts + borrow.ts + user.ts）
- 后端无 API 的（如预约、消息）先用 mock-data.ts 填充
- API 失败时 fallback 到 mock-data
